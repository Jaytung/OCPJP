# 鮮記店商購物平台
## 簡介
此專案為個人使用Java開發練習用
## 功能介紹
- 會員:登入、註冊、修改
- 產品:產品清單、產品分類、產品查詢
- 購物車: 新增/修改/刪除 購物車內容
- 訂單: 訂單明細、歷史訂單查詢
- 結帳: ATM轉帳、EZShip、O'Pay信用卡
